Here you can find some basic automation tests run via Cypress on the website https://computer-database.gatling.io/computers
These were my first automation codes using Cypress, and I'm happy with the results. In the pdf you can find in this repository, you can read about each tests and some findings I did while doing exploratory testing on the web

Instructions on how to run the test

1-Download the folder containing the files
2-Open Cypress
3-Click on "Add project"
4-Drag the folder to Cypress or search for it manually
5-Open the project, and select the "E2E testing" option
6-Select the browser you want to use to run the tests
7-Click on the test you want to run
